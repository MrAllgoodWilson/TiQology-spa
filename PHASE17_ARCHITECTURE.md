## Quick Start

### 1. Setup (One-time)
```bash
# Create and add ROCKET_WRITE_TOKEN secret
gh secret set ROCKET_WRITE_TOKEN

# Enable workflow
gh workflow enable phase17_authorization.yml
```

### 2. Run Operations
```bash
# Status check
gh workflow run phase17_authorization.yml -f operation=status_check

# CI cleanup
gh workflow run phase17_authorization.yml -f operation=ci_cleanup

# Full sync
gh workflow run phase17_authorization.yml -f operation=full_sync -f dry_run=false
```

### 3. Monitor Results
```bash
# View recent runs
gh run list --workflow=phase17_authorization.yml

# Watch current run
gh run watch

# Download artifacts
gh run download <run-id>
```
